# Calculadora em C

Este é um projeto simples desenvolvido para praticar lógica de programação utilizando a linguagem C.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão

## Como funciona
O usuário escolhe uma operação no menu e informa dois números. O programa então calcula e exibe o resultado.

## Tecnologias utilizadas
- Linguagem C

## Objetivo
Projeto criado como prática de estudos no curso de Análise e Desenvolvimento de Sistemas.
